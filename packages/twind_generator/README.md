# twind_generator

Code generator for the `twind` package.

## Usage

This package is automatically used when you add `twind` as a dependency.

```yaml
dev_dependencies:
  twind_generator: ^0.0.1
  build_runner: ^2.4.0
```

See the [twind](https://pub.dev/packages/twind) package for usage instructions.
